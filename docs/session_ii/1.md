# File Handling in Python

File handling is the process of **reading, writing, and managing files stored on disk**.

These skills are extremely important because most real applications work with:

* Configuration files
* CSV data
* JSON APIs
* Log files
* AI datasets
* Model outputs

Python file handling is much simpler.

---

# 1. Opening a File

Use the built-in `open()` function.

```python
file = open("sample.txt", "r")

content = file.read()

print(content)

file.close()
```

However, this is **not recommended**.

If an exception occurs, the file may remain open.

---

# 2. Safe File Handling (`with` statement)

Recommended approach.

```python
with open("sample.txt", "r") as file:

    content = file.read()

    print(content)
```

When execution leaves the `with` block:

* The file is automatically closed
* Even if an exception occurs

Think of it as similar to C#:

```csharp
using(var file = new StreamReader(path))
{
}
```

---

# 3. File Modes

| Mode | Description       |
| ---- | ----------------- |
| `r`  | Read              |
| `w`  | Write (overwrite) |
| `a`  | Append            |
| `x`  | Create new file   |
| `rb` | Read binary       |
| `wb` | Write binary      |
| `r+` | Read and write    |

---

# 4. Writing to a File

```python
with open("notes.txt", "w") as file:

    file.write("Hello Python\n")

    file.write("Learning File Handling")
```

Content:

```text
Hello Python
Learning File Handling
```

`w` overwrites existing content.

---

# 5. Appending to a File

```python
with open("notes.txt", "a") as file:

    file.write("\nNew Line Added")
```

Content:

```text
Hello Python
Learning File Handling
New Line Added
```

---

# 6. Reading Entire File

```python
with open("notes.txt", "r") as file:

    content = file.read()

    print(content)
```

---

# 7. Reading Line by Line

```python
with open("notes.txt", "r") as file:

    for line in file:

        print(line.strip())
```

`strip()` removes extra spaces and newline characters.

---

# 8. Reading All Lines into a List

```python
with open("notes.txt", "r") as file:

    lines = file.readlines()

print(lines)
```

Output:

```python
[
    'Hello Python\n',
    'Learning File Handling\n'
]
```

# 9. File Existence Check

```python
from pathlib import Path

file_path = Path("notes.txt")

if file_path.exists():

    print("File exists")
else:

    print("File not found")
```

---

# 10. Safe Exception Handling

```python
try:

    with open("data.txt", "r") as file:

        print(file.read())

except FileNotFoundError:

    print("File does not exist")

except Exception as ex:

    print(ex)
```

# CSV Basics

CSV = Comma Separated Values.

Example:

```text
id,name,city
1,Someshwar,Pune
2,John,Mumbai
```

Always use Python's built-in `csv` module.

---

# 11. Writing CSV

```python
import csv

employees = [
    [1, "Someshwar", "Pune"],
    [2, "John", "Mumbai"]
]

with open(
    "employees.csv",
    "w",
    newline=""
) as file:

    writer = csv.writer(file)

    writer.writerow([
        "id",
        "name",
        "city"
    ])

    writer.writerows(employees)
```

---

# 12. Reading CSV

```python
import csv

with open(
    "employees.csv",
    "r"
) as file:

    reader = csv.reader(file)

    for row in reader:

        print(row)
```

Output:

```text
['id', 'name', 'city']
['1', 'Someshwar', 'Pune']
['2', 'John', 'Mumbai']
```

---

# 13. CSV Using Dictionaries

Very common.

## Writing

```python
import csv

employees = [
    {
        "id": 1,
        "name": "Someshwar"
    },
    {
        "id": 2,
        "name": "John"
    }
]

with open(
    "employees.csv",
    "w",
    newline=""
) as file:

    writer = csv.DictWriter(
        file,
        fieldnames=[
            "id",
            "name"
        ]
    )

    writer.writeheader()

    writer.writerows(employees)
```

---

## Reading

```python
import csv

with open(
    "employees.csv",
    "r"
) as file:

    reader = csv.DictReader(file)

    for row in reader:

        print(row)
```

Output:

```python
{
 'id': '1',
 'name': 'Someshwar'
}
```

# JSON Basics

JSON = JavaScript Object Notation.

Python:

```python
employee = {
    "id": 1,
    "name": "Someshwar"
}
```

JSON:

```json
{
  "id": 1,
  "name": "Someshwar"
}
```

---

# 14. Write JSON

```python
import json

employee = {
    "id": 1,
    "name": "Someshwar",
    "city": "Pune"
}

with open(
    "employee.json",
    "w"
) as file:

    json.dump(
        employee,
        file,
        indent=4
    )
```

File:

```json
{
    "id": 1,
    "name": "Someshwar",
    "city": "Pune"
}
```

---

# 15. Read JSON

```python
import json

with open(
    "employee.json",
    "r"
) as file:

    employee = json.load(file)

print(employee)
```

Output:

```python
{
 'id':1,
 'name':'Someshwar',
 'city':'Pune'
}
```

# 16. JSON String Conversion

Python → JSON string

```python
import json

employee = {
    "name": "Someshwar"
}

json_string = json.dumps(employee)

print(json_string)
```

Output:

```text

text = '{"name":"Someshwar"}'

print(data)
```

Output:

```python
{
 'name':'Someshwar'
}
```

# Safe Handling Best Practices

## Always use `with`

Good:

```python
with open(
    "file.txt",
    "r"
) as file:

    print(file.read())
```

Avoid:

```python
file = open(
    "file.txt",
    "r"
)

...

file.close()
```

---

## Use explicit encoding

```python
with open(
    "file.txt",
    "r",
    encoding="utf-8"
) as file:

    print(file.read())
```

This avoids cross-platform issues.

---

## Use `Path`

```python
from pathlib import Path

path = Path("data")

path.mkdir(
    exist_ok=True
)

file = path / "sample.txt"

print(file)
```

---

## Handle exceptions

```python
try:

    with open(
        "sample.txt",
        "r"
    ) as file:

        print(file.read())

except FileNotFoundError:

    print("File not found")

except PermissionError:

    print("Permission denied")
```

# Practical Example

```python
import json
import csv

employees = [
    {
        "id": 1,
        "name": "Someshwar",
        "salary": 120000
    },
    {
        "id": 2,
        "name": "John",
        "salary": 90000
    }
]

# Write JSON

with open(
    "employees.json",
    "w"
) as file:

    json.dump(
        employees,
        file,
        indent=4
    )

# Write CSV

with open(
    "employees.csv",
    "w",
    newline=""
) as file:

    writer = csv.DictWriter(
        file,
        fieldnames=[
            "id",
            "name",
            "salary"
        ]
    )

    writer.writeheader()

    writer.writerows(employees)

print("Files created successfully")
```

| -------------- | ------------------ |
