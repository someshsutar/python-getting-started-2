Python has a very rich ecosystem of **third-party libraries** that extend its capabilities far beyond the standard library. These libraries are not included with Python by default, so you install them using **pip (Python Package Installer)**.

---

# 1. What are Third-Party Libraries?

Third-party libraries are **external packages created by the Python community or companies** that provide ready-made functionality for tasks like:

* Data analysis
* Machine learning
* Web development
* API integration
* Automation
* Visualization

They are usually available on **PyPI (Python Package Index)**:
👉 [https://pypi.org](https://pypi.org)

---

# 2. How to Install Third-Party Libraries

## ✔ Using pip (recommended)

### Basic install

```bash
pip install requests
```

### Install specific version

```bash
pip install requests==2.31.0
```

### Upgrade a library

```bash
pip install --upgrade requests
```

### Uninstall a library

```bash
pip uninstall requests
```

---

## ✔ Check installed libraries

```bash
pip list
```

---

## ✔ Install from requirements file

```bash
pip install -r requirements.txt
```

Example `requirements.txt`:

```
requests
numpy
pandas
flask
```

# 3. Popular Third-Party Python Libraries

## 🔹 1. NumPy (Numerical computing)

Used for:

* Arrays
* Mathematical operations
* Scientific computing

### Install:

```bash
pip install numpy
```

### Example:

```python
import numpy as np

arr = np.array([1, 2, 3, 4])
print(arr * 2)
```

---

## 🔹 2. Pandas (Data analysis)

Used for:

* Dataframes
* CSV/Excel handling
* Data cleaning

### Install:

```bash
pip install pandas
```

### Example:

```python
import pandas as pd

data = {"Name": ["A", "B"], "Age": [25, 30]}
df = pd.DataFrame(data)

print(df)
```

---

## 🔹 3. Matplotlib (Data visualization)

Used for:

* Graphs
* Charts
* Data plotting

### Install:

```bash
pip install matplotlib
```

### Example:

```python
import matplotlib.pyplot as plt

x = [1, 2, 3]
y = [10, 20, 30]

plt.plot(x, y)
plt.show()
```

---

## 🔹 4. Requests (HTTP API calls)

Used for:

* Calling REST APIs
* Fetching web data

### Install:

```bash
pip install requests
```

### Example:

```python
import requests

response = requests.get("https://api.github.com")
print(response.status_code)
print(response.json())
```

---

## 🔹 5. Flask (Web framework)

Used for:

* Building web applications
* APIs

### Install:


### Example:

```python

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello World"

app.run()
```

# 4. Typical Workflow of Using a Library

## Step 1: Install library

```bash
pip install requests
```

## Step 2: Import in Python

```python
import requests
```

## Step 3: Use in code

```python
requests.get("https://example.com")
```

# 5. Virtual Environment (Best Practice)

Used to isolate project dependencies.

## Create environment

```bash
python -m venv myenv
```

## Activate environment

### Linux/Mac:

```bash
source myenv/bin/activate
```

### Windows:

```bash
myenv\Scripts\activate
```

## Install packages inside it

```bash
pip install numpy pandas
```
