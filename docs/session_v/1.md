Python comes with a rich set of **standard libraries** and **built-in functions** that allow you to perform common tasks without installing additional packages.

---

# 1. Python Standard Library

The **Python Standard Library** is a collection of pre-written modules that come with every Python installation. These modules provide functionality for file handling, mathematics, networking, operating system interaction, dates, JSON processing, regular expressions, and much more.

To use a standard library module, import it:

```python
import math
print(math.sqrt(25))
```

Output

```
5.0
```

---

## Common Standard Library Modules

| Module        | Purpose                          |
| ------------- | -------------------------------- |
| `math`        | Mathematical operations          |
| `random`      | Random number generation         |
| `datetime`    | Date and time handling           |
| `os`          | Operating system operations      |
| `sys`         | Python interpreter interaction   |
| `json`        | Read/write JSON data             |
| `csv`         | Read/write CSV files             |
| `statistics`  | Statistical calculations         |
| `collections` | Specialized container data types |
| `itertools`   | Efficient looping                |
| `pathlib`     | File path handling               |
| `re`          | Regular expressions              |
| `time`        | Time-related functions           |
| `shutil`      | File copying and moving          |
| `logging`     | Logging application events       |

---

## Example 1: math module

```python
import math

print(math.sqrt(64))
print(math.factorial(5))
print(math.pi)
```

Output

```
8.0
120
3.141592653589793
```

---

## Example 2: random module

```python
import random

print(random.randint(1, 10))
print(random.choice(["Apple", "Banana", "Orange"]))
```

---

## Example 3: datetime module

```python
from datetime import datetime

today = datetime.now()

print(today)
print(today.year)
print(today.month)
```

---

## Example 4: os module

```python
import os

print(os.getcwd())      # Current directory
print(os.listdir())     # List files
```

---

## Example 5: json module

```python
import json

student = {
    "name": "John",
    "age": 20
}

json_data = json.dumps(student)

print(json_data)
```

Output

```
{"name": "John", "age": 20}
```

# 2. Built-in Functions

Built-in functions are available automatically. They do **not require importing** any module.

Example:

```python
print("Hello")
```

---

## Common Built-in Functions

| Function      | Purpose                       |
| ------------- | ----------------------------- |
| `print()`     | Display output                |
| `input()`     | Read user input               |
| `len()`       | Length of object              |
| `type()`      | Object type                   |
| `int()`       | Convert to integer            |
| `float()`     | Convert to float              |
| `str()`       | Convert to string             |
| `list()`      | Create list                   |
| `tuple()`     | Create tuple                  |
| `set()`       | Create set                    |
| `dict()`      | Create dictionary             |
| `range()`     | Generate sequence             |
| `sum()`       | Sum of numbers                |
| `max()`       | Largest value                 |
| `min()`       | Smallest value                |
| `sorted()`    | Sort iterable                 |
| `abs()`       | Absolute value                |
| `round()`     | Round number                  |
| `enumerate()` | Add index while iterating     |
| `zip()`       | Combine iterables             |
| `any()`       | True if any element is True   |
| `all()`       | True if all elements are True |

---

## Example: len()

```python
fruits = ["Apple", "Banana", "Orange"]

print(len(fruits))
```

Output

```
3
```

---

## Example: sum()

```python
numbers = [10, 20, 30]

print(sum(numbers))
```

Output

```
60
```

---

## Example: max() and min()

```python
marks = [65, 80, 72, 91]

print(max(marks))
print(min(marks))
```

Output

```
91
65
```

---

## Example: sorted()

```python
numbers = [5, 2, 8, 1]

print(sorted(numbers))
```

Output

```
[1, 2, 5, 8]
```

---

## Example: enumerate()

```python
fruits = ["Apple", "Banana", "Orange"]

for index, fruit in enumerate(fruits):
    print(index, fruit)
```

Output

```
0 Apple
1 Banana
2 Orange
```

---

## Example: zip()

```python
names = ["John", "Alice", "Bob"]
marks = [80, 90, 85]

for name, mark in zip(names, marks):
    print(name, mark)
```

Output

```
John 80
Alice 90
Bob 85
```

---

## Example: any() and all()

```python
values = [True, False, True]

print(any(values))
print(all(values))
```

Output

```
True
False
```

---

# Difference Between Standard Library and Built-in Functions

| Built-in Functions                    | Standard Library                   |
| ------------------------------------- | ---------------------------------- |
| Available automatically               | Must be imported                   |
| No `import` required                  | Requires `import`                  |
| Examples: `print()`, `len()`, `sum()` | Examples: `math`, `random`, `os`   |
| Used for common operations            | Used for specialized functionality |

---

# Demonstration Program

```python
import math
import random
from datetime import datetime

numbers = [5, 10, 15, 20]

# Built-in functions
print("Length:", len(numbers))
print("Sum:", sum(numbers))
print("Maximum:", max(numbers))
print("Sorted:", sorted(numbers))

# Standard library
print("Square Root of 25:", math.sqrt(25))
print("Random Number:", random.randint(1, 100))
print("Current Date & Time:", datetime.now())
```

### Sample Output

```
Length: 4
Sum: 50
Maximum: 20
Sorted: [5, 10, 15, 20]
Square Root of 25: 5.0
Random Number: 67
Current Date & Time: 2026-06-27 15:40:10.123456
```
